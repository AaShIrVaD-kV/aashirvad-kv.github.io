## ðŸŒ How to Deploy to GitHub Pages

1.  **Create a New Repository** on GitHub (e.g., `my-portfolio`).
2.  **Upload Files**:
    - Upload `index.html`, `style.css`, `script.js` (and your PDF resume) to this repository.
3.  **Enable GitHub Pages**:
    - Go to your Repository **Settings**.
    - Click on **Pages** in the left sidebar.
    - Under "Build and deployment" > "Branch", select `main` (or `master`) and hit **Save**.
4.  **Done!** GitHub will provide you with a URL (e.g., `https://yourusername.github.io/my-portfolio/`) where your site is live.
